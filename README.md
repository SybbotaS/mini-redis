# Mini Redis

**Mini Redis** — это упрощённая реализация Redis, написанная на Go. Проект демонстрирует разработку TCP-сервера, конкурентную обработку клиентских подключений с использованием goroutines, потокобезопасное хранение данных в памяти с помощью `sync.RWMutex` и разделение приложения на независимые модули.

## Возможности

- TCP-сервер
- Обработка нескольких клиентов одновременно
- Потокобезопасное in-memory хранилище
- Собственный текстовый протокол
- Консольный клиент
- Базовые операции с ключами

## Поддерживаемые команды

| Команда | Описание |
|----------|----------|
| `PING` | Проверка доступности сервера |
| `SET <key> <value>` | Сохранить значение |
| `GET <key>` | Получить значение |
| `DEL <key>` | Удалить ключ |
| `EXISTS <key>` | Проверить существование ключа |

## Что реализовано

- разработан TCP-сервер для обработки клиентских подключений;
- реализована конкурентная обработка клиентов с использованием goroutines;
- создано потокобезопасное хранилище данных на основе `map` и `sync.RWMutex`;
- разработан собственный текстовый протокол обмена сообщениями;
- реализован консольный клиент для взаимодействия с сервером;
- выполнено разделение проекта на независимые модули (`server`, `protocol`, `storage`).

## Планы по развитию

- поддержка TTL для ключей;
- автоматическое удаление просроченных ключей;
- сохранение данных на диск;
- автоматическая загрузка данных при запуске;
- graceful shutdown;
- расширение набора поддерживаемых команд;
- написание unit-тестов.

## Цель проекта

Проект разработан для практики сетевого программирования на Go, изучения конкурентности, работы с TCP-соединениями, синхронизации доступа к данным и построения серверных приложений с чистой архитектурой.
